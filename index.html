Skills

Javascript

É uma linguagem de programação que torna páginas da internet interativas para os clientes. Javascript, HTML e CSS são as 3 principais linguagens do desenvolvimento web

HTML5

Significa Hyper Text Markup Language, que é uma linguagem de marcação responsável por estruturar elementos na página de forma semântica(as tags carregam um significado relacionado a seu conteúdo).

CSS3

É uma sigla para Cascade Style Sheets. Essa linguagem é resonsável pela formatação dos conteúdos, alterando dimensões, cores, entre outras propriedades. Também é responsável por tornar as páginas responsivas, adaptando o conteúdo aos mais diversos dispositivos nos quais a página poderá ser acessada.

Git

É um sistema de controle de versões open source. É muito útil para o desenvolvimento de software, pois permite ao profissional manter um controle das alterações aplicadas em um projeto sem a necessidade de um backup.

GitHub

Uma plataforma de hospedagem de código-fonte, que permite ter o controle de versões utilizando a tecologia Git. Permite que os usuários contribuam nos código uns dos outros. Dessa maneira, se tornou uma espécie de rede social para programadores.
